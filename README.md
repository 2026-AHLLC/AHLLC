# AH LLC

AI-powered business growth platform built with Next.js 15.

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- shadcn/ui
- Framer Motion
- Vercel AI SDK
- Supabase
- Resend

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit:

http://localhost:3000

## Production

```bash
npm run build
npm start
```

## Features

- AI Consultant
- Business Automation
- Portfolio
- Local SEO
- Contact Forms
- Responsive Design
- Dark Theme
